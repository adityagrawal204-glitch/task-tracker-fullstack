# Task Tracker

A full-stack task management application developed using Spring Boot and Vanilla JavaScript.

## Features

- Create Tasks
- View Tasks
- Update Tasks
- Delete Tasks
- RESTful API Integration
- H2 In-Memory Database
- Responsive Frontend

## Tech Stack

### Backend
- Java 17
- Spring Boot
- Spring Data JPA
- Hibernate
- H2 Database

### Frontend
- HTML
- CSS
- JavaScript

## Architecture

Frontend → REST API → Service Layer → Repository Layer → Database
